## AniTime
Anime schedule, korean subtitle for iOS with SwiftUI + Combine and MVVM architecture

## App Features
- [x] Anime schedule List
- [ ] Anime Detail
- [ ] Recent Subtitle
- [ ] Bookmarks
- [ ] Complete Anime List
